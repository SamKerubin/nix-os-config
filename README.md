# My NixOS config files!!
Be aware, this config might be a bit sketchy as i did this while learning the nix programming language as well

Anyways, you need to download this repo, move the `nix-os-config` directory to your home root, cd to it and then execute: `sudo nixos-rebuild switch --flake ~/nix-os-config#sam`

Theres another way of doing this without downloading the repo, yes... But i honestly forgot how to so im just pointing this out as i might be the only one using this repo :P
